## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Initialise Repo
To initialise this repository, open your terminal of choice and run the following command:

```
yarn
```

### Build
To run this project in development mode, run the following command, which creates a development location called **/dev**

```
Yarn run dev-build
```

On success, your console will tell you it the build was successful has compiled and that **./dist** was created.
